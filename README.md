# low code

组件区域+编辑区+配置区

组件区使用mockData 实现组件的渲染  
编辑区每次监听到拖拽后，发出请求，请求组件资源  
配置区 开发一系列 配置组件

组件区实现拖拽，一个组件模块创建两个div元素，一个固定定位，一个相对定位，当监听到对每个组件初始化一个draggable的模块，当拖拽时相对定位，然后 在编辑区使用Interactjs的dropzone，监听目标元素的drop，监听到后实现数据变化。
